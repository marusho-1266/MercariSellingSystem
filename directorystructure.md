# ディレクトリ構成（※本記載は記入例です-プロジェクトに合わせて内容を更新してください-）

以下のディレクトリ構造に従って実装を行ってください：

```
/
├── src/                          # GASのソースディレクトリ
│   ├── appsscript.json           # GASソース管理設定ファイル
├── .git/                         # Gitリポジトリ
├── .cursor/                      # Cursor設定
├── .clasp.json                   # GASソース管理設定ファイル
├── .claspignore                  # GASソース管理除外設定
└── .gitignore                    # Git除外設定
```
